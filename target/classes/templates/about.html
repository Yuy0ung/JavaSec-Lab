关于 Java Security Lab

项目简介

Java Security Lab 是一个专门用于演示和学习Java Web安全漏洞的实验室环境。本项目旨在帮助开发者、安全研究人员和学生深入理解常见的Web安全漏洞，掌握攻击原理和防护方法。

涵盖的漏洞类型

  • SQL注入 (SQLi): 联合查询注入、布尔盲注、时间盲注等
  • 跨站脚本 (XSS): 反射型XSS、存储型XSS、DOM型XSS
  • 命令执行 (RCE): 系统命令注入、代码执行漏洞
  • 服务端请求伪造 (SSRF): 内网探测、云服务攻击
  • XML外部实体注入 (XXE): 文件读取、SSRF攻击
  • 文件上传漏洞: 绕过技术、Webshell上传

技术栈

  • Spring Boot 2.7.18
  • Spring Data JPA
  • Thymeleaf 模板引擎
  • H2 内存数据库
  • Bootstrap 5 前端框架
  • Font Awesome 图标
  • Maven 构建工具
  • Java 14+

学习目标

  • 理解常见Web安全漏洞的成因和危害
  • 掌握各种攻击技术和payload构造方法
  • 学习有效的安全防护措施和最佳实践
  • 提升代码安全审计和漏洞挖掘能力
  • 培养安全开发意识和习惯

重要声明

  • 本实验室仅用于教育和学习目的
  • 请勿在生产环境中部署或使用本项目代码
  • 请勿将学到的攻击技术用于非法用途
  • 使用者需承担相应的法律责任

使用建议

  1. 建议在隔离的测试环境中运行本实验室
  2. 每个漏洞模块都包含详细的说明和示例
  3. 尝试不同的攻击payload，观察系统响应
  4. 学习每种漏洞的防护方法和安全编码规范
  5. 可以查看源代码了解漏洞的具体实现
